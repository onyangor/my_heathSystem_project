Running the project

steps

Create the virtual environment:
bash
python -m venv venv

Activate the virtual environment:
bash
venv\Scripts\activate

Install the requirements:
bash
pip install -r requirements.txt


Then run the app:

bash

uvicorn app.main:app --reload
